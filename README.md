## E-Commerce(옷 쇼핑몰) 프로젝트
- 본 프로젝트는 3계층 아키텍처를 기반으로 설계 및 구현되었습니다.

## 개발 스택
- 언어 : Java(17)
- 프레임워크 : Spring Boot(3.3.4)
- ORM : Spring Data Jpa
- 데이터베이스 : MySQL(8.0.32)
- 테스트 : JUnit5, Mockito
